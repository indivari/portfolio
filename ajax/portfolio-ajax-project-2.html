Project Info:

A server-side CRUD app called "Trip-Finder" to find and rate interesting trip locations in Australia.The main features include:

  • Users can add, edit, delete or view trip locations.
  • Users can signup, login and logout from the system.
  • Keeps track of user sessions.
  • The user can filter trip locations based on the major cities.
  • The user can rate trip locations.
  • Displays trip locations based on the descending order of user rating.

Project Details:

  • Technologies:HTML, CSS, Javscript, Ruby, Sinatra & PostgreSQL
