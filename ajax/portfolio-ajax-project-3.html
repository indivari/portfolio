Project Info:

A single page web app called "Servo app" shows the petrol stations closer to your current location. It uses Google maps API to mark petrol stations locations. Also it shows some useful petrol station statistics such as total number of stations and group summary by station owners.

Project Details:

  • Technologies:Node, Javascript, Express Js, Ajax, PostgreSQL, RESTful API, Google maps API
