Project Info:

Moodbooster is a platform where users can find insipirational quotes that match their interests. Main features include:

  • User authentication using Facebook and Github with OAuth/passportJS
  • Users can post their quotes.
  • Users can tag quotes to different groups
  • Users can vote on quotes by others.
  • Users can Change and delete their own quotes
  • Users can search for quotes by tags

Project Details:

  • Technologies:HTML5, CSS, JavaScript, MongoDb, Axios, React, RESTful APIs, Node, ExpressJS, OAuth
