Project Title 1

Project Info:

A single page online game app to play the classic game 'Tic-Tac-Toe'. Customized according to a kids theme with bright colors and kids friendly characters. It provides the options to "play with a friend" or "play with computer". It uses the Min-Max game logic to determine the winner while playing with the computer.

Project Details:
